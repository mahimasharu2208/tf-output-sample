# tf-output-sample

This example shows how to define and use `output` blocks in Terraform to expose resource attributes.

## Files
- `main.tf` – Defines some basic resources or values.
- `outputs.tf` – Outputs the defined values.

## Usage

```bash
terraform init
terraform apply
```
